以下是ES6最常用的几个语法，基本上学会它们，我们就可以走遍天下都不怕啦！我会用最通俗易懂的语言和例子来讲解它们，保证一看就懂，一学就会。

最常用的ES6特性


1. let 局部变量, const 常量,



2. class 类, extends 继承, super 父类的this指向,



3. arrow functions 函数声明,

ES5
function(i){ return i + 1; }

ES6
(i) =>  + 1

ES5
function(x, y) {
    x++;i
    y--;
    return x + y;
}

ES6
(x, y) => {x++; y--; return x+y}



4. template string 模版, 用反引号（`）来标识起始，用${}来引用变量，而且所有的空格和缩进都会被保留在输出之中

ES5
$("#result").append(
  "There are " + basket.count + " " +
  "items in your basket, " +
  "" + basket.onSale +
    " are on sale!"
);

ES6
$("#result").append(`
  There are ${basket.count} items
   in your basket, ${basket.onSale}
  are on sale!
`);



5. destructuring 解构, ES6允许按照一定模式，从数组和对象中提取值，对变量进行赋值，这被称为解构

ES5
let cat = 'ken'
let dog = 'lili'
let zoo = {cat: cat, dog: dog}
console.log(zoo)  //Object {cat: "ken", dog: "lili"}

ES6
let cat = 'ken'
let dog = 'lili'
let zoo = {cat, dog}
console.log(zoo)  //Object {cat: "ken", dog: "lili"}

反过来可以这么写：
let dog = {type: 'animal', many: 2}
let { type, many} = dog
console.log(type, many)   //animal 2



6. default 默认值,

function animal(type = 'cat'){
    console.log(type)
}
animal();



7. rest arguments 参数

function animals(...types){
    console.log(types)
}
animals('cat', 'dog', 'fish') //["cat", "dog", "fish"]
